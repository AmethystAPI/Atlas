# Atlas 

Atlas is a Minimap mod for `1.21.0.3` on Minecraft Bedrock Edition using the [AmethystAPI](https://github.com/FrederoxDev/Amethyst) modding platform. The mod is developed by @FrederoxDev and @theaddonn.

## Controls and Usage

To zoom in and out with the minimap, hit the `+` and `-` keys. The mod is also shipped with a helper resource pack for now which should be placed into global resources when using the mod.
